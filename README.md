**Client:** React, Redux,

**Server:** firebase , Stripe

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/Mukul-Singhal/Crown_Clothing
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
